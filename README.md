### Option 2: Netlify (Free & Easy)

1. Go to [netlify.com](https://www.netlify.com)
2. Sign up for a free account
3. Drag and drop your entire `Cursor_Portfolio` folder
4. Your site will be live immediately with a random URL
5. You can customize the domain name in settings

### Option 3: Vercel (Free & Easy)

1. Go to [vercel.com](https://www.vercel.com)
2. Sign up for a free account
3. Click "New Project"
4. Import your GitHub repository or upload files
5. Deploy instantly

### Option 4: Traditional Web Hosting

1. Purchase web hosting (e.g., Bluehost, HostGator, etc.)
2. Upload all files via FTP or hosting control panel
3. Place files in the `public_html` or `www` folder
4. Your site will be live at your domain

## Customization

- **Colors**: Edit the orange color `#f58327` in `styles.css` to change the theme
- **Content**: Edit `index.html` to update text and information
- **Images**: Replace images in the `imgs/` folder

## Notes

- All images should be in the `imgs/` folder
- Keep the folder structure intact for proper image loading
- The website uses Google Fonts (Poppins) - requires internet connection
